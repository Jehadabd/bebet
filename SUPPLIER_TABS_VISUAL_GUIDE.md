## 🎨 الألوان المستخدمة

### فواتير النقد 🧾
- **اللون الرئيسي:** أزرق (`Colors.blue`)
- **الخلفية:** أزرق فاتح (`Colors.blue[50]`)
- **النص:** أزرق داكن (`Colors.blue[700]`)

### فواتير الدين 💳
- **اللون الرئيسي:** أحمر (`Theme.colorScheme.error`)
- **الخلفية:** أحمر فاتح (`Colors.red[50]`)
- **النص:** أحمر داكن (`Colors.red[700]`)

### سندات القبض 💰
- **اللون الرئيسي:** أخضر (`Colors.green[700]`)
- **الخلفية:** أخضر فاتح (`Colors.green[50]`)
- **النص:** أخضر داكن (`Colors.green[700]`)

## 📐 الأبعاد والمسافات

### البطاقات
- **الارتفاع:** تلقائي (حسب المحتوى)
- **الحواف الدائرية:** 12px
- **الظل:** elevation: 2
- **الحشو الداخلي:** 16px

### الأيقونات
- **الحجم:** 28px
- **الحشو:** 12px
- **الخلفية:** دائرية 12px

### النصوص
- **العنوان:** titleMedium, bold
- **التاريخ:** bodyMedium, grey
- **المبلغ:** titleMedium, bold, ملون
- **التسمية:** bodySmall, bold

### المسافات
- **بين البطاقات:** 12px
- **بين العناصر:** 4-8px
- **الحشو الخارجي:** 8px عمودي

## 🎯 الخلاصة البصرية

```
الشاشة = معلومات المورد + شريط التبويبات + قائمة المعاملات

شريط التبويبات = 3 تبويبات (نقد، دين، سندات)

كل تبويب = قائمة بطاقات مرتبة

كل بطاقة = أيقونة + معلومات + مبلغ + تسمية

الضغط على بطاقة = حوار تفاصيل منظم
```
